Troubleshooting
  • This must be opened over HTTPS (GitHub Pages gives you this automatically) — won't work on file://.
  • On iPhone: Add this page to your Home Screen first (Share → Add to Home Screen), then open it FROM the home screen icon — Safari only allows push notifications for installed/home-screen web apps, not regular tabs.
  • On Android/Desktop Chrome: works directly in the browser tab, no install needed.
  • If you ever clear browser data or reinstall, you'll need to resubscribe (repeat this page) and update subscription.json.
